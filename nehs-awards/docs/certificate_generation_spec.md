# 獎狀產生規格

## 目標

建立一套可以從比賽名次資料產生獎狀 PDF 的規則。網站階段之後再做；目前先確認 PDF 背景、字體、欄位與座標可行。

目前 Template 1 到 Template 5 的位置已由使用者確認，後續開發請保持這份文件記錄的座標，除非使用者再次要求調整。

## 字體

- 強制使用 `edukai-5.1_20251208.ttf`
- 在程式中註冊為 `EduKai`
- 中文與英文目前都使用同一個字體

## 背景

目前可用兩類背景：

- 原始 Word 匯出模板：
  - `PDF/114運動會獎狀.pdf`
  - `PDF/114運動會獎狀_接力中學.pdf`
  - `PDF/114運動會獎狀_接力雙語_20251111.pdf`
  - `PDF/114運動會獎狀_接力雙語11以上_20251111.pdf`
- 掃描獎狀背景：
  - `PDF/SCAN3679.pdf`

`SCAN3679.pdf` 是掃描背景，包含邊框、校名、國旗、校長簽名與紅色印章。因為背景已經有校長簽名與印章，所以文字區必須集中在中上半部，日期目前放在紅色印章下緣附近。

## SCAN3679 目前座標規則

PDF 使用 A4 直式座標，單位是 `pt`：

- 頁面左下角是 `(0, 0)`
- 往右：`x` 增加
- 往上：`y` 增加
- 往下移動：`y` 減少
- A4 約為 `595 x 842 pt`

## Name / EName 拆分規則

雙語部資料若網站 `Name` 同時含中文與英文，應拆成：

- `Name`：中文姓名
- `EName`：英文姓名

建議規則：

- 中文字元連續區段視為 `Name`
- 英文字母、逗號、連字號、空白組成的區段視為 `EName`
- 若只有中文，`EName` 留空
- 若只有英文，保留在 `EName`，`Name` 需人工確認

## 後續網站流程草案

1. 抓取或匯入比賽名次資料。
2. 依比賽名稱分類。
3. 依年級分類。
4. 使用者選擇比賽與年級。
5. 顯示該年級名單與名次。
6. 勾選第 1 到第 5 名。
7. 依學生部別與項目類型選擇模板。
8. 產生獎狀 PDF。

## 注意事項

- `SCAN3679.pdf` 是掃描圖，不是可編輯模板；所有文字都必須用座標疊上去。
- 接力模板內容較多，正式版應獨立調整，不能完全套用單人模板位置。
- 長英文姓名需要自動縮字或換行。
- 日期位置若固定在印章下方，可能會靠近底部邊框；正式版可做成可調參數。
